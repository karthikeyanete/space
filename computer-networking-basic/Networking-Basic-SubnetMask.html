Subnet Masking

  • Network - 1
  • Host - 0

    • What is SubnetMask ?

    Subnet mask is the combination of network and host. It is used to determine the Network ID and Host bit in IP address

    A computer find the Network Id and Broadcast ID with help of the IP address and Subnetmask

    Default SubnetMask is also called as Classful

    Network ID

    Donot disturbed in Network Bits all Host Bits are 0

    First ID

    Donot disturbed in Network Bits all Host Bits are 0 except Last bit 1

    Last ID

    Donot disturbed in Network Bits all Host Bits are 1 except Last bit 0

    Broadcast ID

    Donot disturbed in Network Bits all Host Bits are 1
