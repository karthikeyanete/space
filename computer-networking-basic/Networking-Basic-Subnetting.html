What is Subnetting ?

Dividing the Major Network into multiple Small network according to the requirements. Originally all IP address is classfull. If IP address having more than the default network bits is called classless. Classfull IP address is transfered into Classless this method is Called subnetting

Classfull → Subnetting → Classless

What is the benefit of subnetting ?

Avoiding the wastage of IP addresss and it is utilized based on the requirements

How to do Subnetting ?

Example : In an Organisation there is six department for that the Subnetting is to be done in the Given IP adress

Step 1:

Borrow New network bits From the Host portion according to the requirement

  • Example IP : Class C => 192.1.1.0

    • NetworkNetworkNetworkHost
    192.0.0. 00000 0 00
    --24 Default Network Bits--
    <--->  8 Host Bits  <--->
    ---24 Network Bits + 3 New Network Bits---
    --- 5 Host Bits ---
  • If one bit borrowed from 25th bit => 2^1 => 2 Combinations
  • If two bit borrowed from 25 th and 26th => 2^2 => 4 Combinations
  • If three bit borrowed from 25th,26th and 27th => 2^3 => 8 Combinations

    • Our Requirement is 6 subnets so 3 bits borrowed

    Formulae

  • 2^n n=> Number of Subnets
  • Requirement => 6
  • 2^1 = 2
  • 2^2 = 4
  • 2^3 = 8
  • Now 3 Bits are New Network Bits. and now it is called as Classless

    Finding New SubnetMask

  • Default SubnetMask =>24 =>255.255.255.0
  • New SubnetMask =>27 =>255.255.255.224
