Internet Protocol Version 6

Internet Protocol Version 6 is a Numeric label that is used to identify the device connected in an IPv6 computer network.Internet Protocol version 6(IPV6) is sixth version of internet protocol,It consists of 128 bit alphanumeric string, each block consists of 16 bits

Developed by IANA (Internet Assigned Number Authority)
