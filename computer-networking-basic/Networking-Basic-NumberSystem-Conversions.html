Why number System Conversions

In digital hardware the natural numbering system is base 2 that is 0’s and 1’s but representing in the binary system makes huge bits to be used, so in practical it is not possible, followed by we use Hexa Decimal Number System

For Example: 1110 1111 (Binary) it is 8 bit which is equivalent to E F (Hexa Decimal) and 239 (Decimal)

If in any Number System the input is given during computation process the conversion is done to the Binary and the result is converted to required format.
