    Network topology

    It is the arrangement of the elements of the network. also called as virtual structure of the network. The network topologies are

  • Bus topology
  • Star topology
  • Ring topology
  • Mesh topology

    • Bus topology

    It is a topology for a LAN in which all the nodes are connected to a single cable. the wire used to connect the computer is called as bus. all the computers are connected to bus and its end is connected with the terminator.

    Advantages

  • It is easy to connect and required less cables
  • Low cost,simple,reliable and works well for small networks
  • If one node fails it will not affect other

    • Disadvantages

  • Entire network will be failed if any fault in the main cable or T connector
  • Slow with many computers and perfomance is less

    • Star topology

    It is a topology for a LAN in which all the nodes are connected to a center point like a hub or a switch. It takes more cable than bus

    Advantages

  • If one node or connection breaks it doesnot affect other computers
  • During addition or removal of the network the network will not get disturbed

    • Disadvantages

  • Expensive network each host is connected to the hub or router through seperate cables
  • If router or hub fails then the total network will get failed

    • Ring topology

    It is a topology which each node connected to other node and last computer is connected to the first computer. The computers are connected in the ring form and each computer transmits the data which recieved from previous computer. The data flows in the form of Ring in one direction.

    Advantages

  • Performs better then bus topology. Point to point communication
  • Does not require a central node to manage all the networks

    • Disadvantages

  • Adding or removal of the node will affect whole the network
  • Communication delay is occured according to the number of computers
  • If one computer fails then the hole network will get failed

    • Mesh topology

    It is a topology in which each node connected to each other node.The data is send directly from one device to another device

    Disadvantages

  • Difficult to configure and maintain
  • More cables are used it is expensive
