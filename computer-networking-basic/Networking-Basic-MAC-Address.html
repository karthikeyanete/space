Media Acess Control (MAC)

  • It is a hardware identification number that uniquely identifies the device on the network
  • It is a global unique identifier, also called as hardware or physical address
  • 6-byte (48-bits) in length
  • Hexa Decimal Format

    • Example Mac Address

  • 00 - AB - CD - D8 - CA - FC
  • 00000 00000 - 1010 1011 - 1100 1101 - 1101 1000 - 1100 1010 - 1111 1100
