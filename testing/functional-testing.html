Functional Testing

Functional testing is the process of giving input and examining the expected result. It is a type of Black box testing. The term functional testing doesn't denote function ar a class, It tests the function of the whole product or service.

The functional testing has many types they are

  • Unit testing
  • Integration testing
  • System testing
  • Sanity testing
  • Smoke testing
  • Interface testing
  • Regression Testing
  • Beta Testing
